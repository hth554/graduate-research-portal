🔑 首次使用需要设置 GitHub Token

  1. 访问 GitHub Token 页面
  2. 点击 "Generate new token (classic)"
  3. Token 名称：课题提交系统
  4. 权限至少勾选：repo（完全控制仓库）
  5. 点击 "Generate token" 并复制
  6. 粘贴到下方输入框：

Token 仅存储在您的浏览器本地，不会上传到任何服务器。
